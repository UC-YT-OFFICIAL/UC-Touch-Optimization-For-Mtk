UC Touch Optimization For Mali (MTK)

📌 Overview

UC Touch Optimization is a Magisk module designed to improve touch responsiveness, UI smoothness, and frame stability on devices powered by Mali GPUs (MediaTek platforms).

This module tweaks SurfaceFlinger timing parameters and GPU behavior to provide a smoother and more responsive user experience.

---

⚙️ Features

- Improved touch responsiveness
- Reduced frame latency
- Better frame pacing
- Optimized SurfaceFlinger timing
- Mali GPU scheduling tweaks
- Smoother scrolling and UI animations

---

📱 Compatibility

- MediaTek devices (MTK)
- Mali GPU required
- Android 10+
- Magisk v20.4+
- ksu required - meta module 
---

📦 Installation

1. Download the module ZIP
2. Open Magisk app
3. Go to Modules → Install from storage
4. Select the ZIP
5. Reboot

---

⚠️ Notes

- Results may vary by device
- May increase battery usage slightly
- Not guaranteed to improve performance on all ROMs
